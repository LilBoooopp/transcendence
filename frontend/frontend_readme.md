# structure

## components
All Components are stored in the components folder.

## SamplePage
Sample Pages go here, these are not linked to the Backend for now.

## styles.ts
In this file the basic style settings for classes are stored.

## tailwind.config.js
In this file the color pallet is defined as well as the fonts.

# icon library
For the Icons I used the lucide-react library.
It offers over 1.5k icons.
To look through the icons and search for a fitting one go to:
https://lucide.dev/

# usage
There is no special usage.
To access the SamplePages use the specified url which you can find in the App.tsx file.
